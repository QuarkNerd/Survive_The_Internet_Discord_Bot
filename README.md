# Survive the Web Discord bot

Discord bot game based on twisting your friends words

## Play

This Bot is not currently hosted, to play you must install and register it yourself

## Installation Steps (if you want to mess with the code)

1. Clone repo
2. Run `npm install`
3. Register a new discord bot. Go [here](https://www.sitepoint.com/discord-bot-node-js/).
4. Add Discord token in a `.env` file under the name "TOKEN"
5. Run `npm start`
6. Interact with the Discord bot
